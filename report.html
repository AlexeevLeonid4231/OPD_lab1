Отчет по лабораторной работе № 1

«Определение видов программистов и ИТ, ПВК для определенного вида деятельности»

Студент

ФИО: Алексеев Леонид Алексеевич группа: P3323

1. Цель и задачи лабораторной работы

Научиться разрабатывать системы в проектной деятельности, формировать рабочие группы и распределять рабочие роли в команде.

  • Разработать портал на HTML/CSS/JS.
  • Реализовать открытые и закрытые разделы.
  • Смоделировать роли: администратор, пользователь, эксперт, консультант.
  • Подготовить демонстрацию на статическом хостинге.

2. Методика проведения исследования

Система реализована как статический веб-портал, разворачиваемый на платформе статического хостинга. Данные о профессиях хранятся в JSON, пользовательские действия сохраняются в локальном хранилище браузера. Для оценок используется среднее арифметическое.

Формулы: средняя оценка по профессии = (Σ оценок) / n.

3. Результаты

  • Каталог профессий с фильтрами и оценками экспертов.
  • Регистрация, вход, роли и сценарии.
  • Консультации пользователей и кабинет консультанта.
  • Панель администратора с локальным управлением сущностями.

4. Выводы

Все задания выполнены. Возникали сложности с моделированием серверных функций в статическом окружении, решено сохранением в localStorage.
